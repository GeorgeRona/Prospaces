
  # ProSpaces CRM

  This is a code bundle for ProSpaces CRM. The original project is available at https://www.figma.com/design/0NIWxaPFaVcCTCDFcePevZ/ProSpaces-CRM.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Adding a Contributor

  To add a contributor (or any user) to your organization:

  1. Log in as an **Admin** or **Super Admin**.
  2. Open the **Admin** menu in the left navigation sidebar.
  3. Select **Users** from the submenu.
  4. Click the **Add User** (or **Invite User**) button.
  5. Fill in the user's name and email, and select **Contributor** as their role.
  6. Choose a delivery method (email invite or temporary password) and submit.

  The **Contributor** role grants the user read and write access to core CRM modules (contacts, tasks, notes, appointments, opportunities, bids, quotes, inventory, etc.) while restricting access to administrative areas and preventing record deletion.
